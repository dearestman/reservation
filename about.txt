Дананя система дает возможность резервировтаь помещения, переговорные, актовые залы и т.д.
Также ее можно использовать в паре с Api team viewer.
Также система должна направлять уведомление пользователю:
    - Когда пользователь создал встречу;
    - Когда пользователь изменил информацию о встрече (также когда приглосил людей);
    - Когда пользователь удалил встречу;
    - За час до встречи (напоминание во всех учавствующих встречах)

В данной системе есть права пользователй:
-администратор:
    - Имеет отдельный функционал для администрирования;
        - создает новых пользователей;
        - изменяет информацию о пользователе;
        - удаляет пользователей;
        - смотрит лог действий пользователей.
        - назначает права пользователям:
            - Гость - может смотреть только свой список встреч, где он учавствует;
            - Пользователь системы - может смотреть свой список встречь, а также создавать новые и редактировать созданные встречи
        - Может изменять и создавать встречи, а также их удалять
        - Может добавлять, изменять, удалять тип встреч;
- пользователь
    - Может создавать новые встречи
    - Может редактировать свои встречи
    - Может удалять свои встречи
    - Может добавлять пользователей в свои встречи
    - Может удалять пользователей в свои встречи
    - Может оставлять комментарии в приглашенные встречи
    - Может назначать права гостям своих втсреч, чтобы они могли ее администрировать
- гость
    - может смотреть информацию о встречи, на которую пригласили
    - если есть права на редактирование встречи (Администратор встречи) - может ее редактировать
